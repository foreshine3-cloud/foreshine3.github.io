搖控柵欄升降器操作說明書

■內含配件:

  1. 主機體 x 1
  2. 遙控器 x 1
  3. 鋁旗桿 x 1
  4. 插銷 x 1
  5. 旗幟(停) x 1
  6. 三腳架 x 1
  7. 電池 x 1
  8. 充電器 x 1
  9. 帆布袋 x 1組（4件）

■安裝說明

  1. 展開三角架並立置。
  2. 將主機體插入三角架並鎖緊。
  3. 將鋁旗桿插入主機體鐵管內，並以插銷固定。
  4. 將主機體電源線接電池，紅線(+正電)接正極(紅端)，黑線(-負電)接負極(黑端)。
  5. 配置LED，將LED電源線接至主機體電控箱下方的電源插座。

■操作說明:

      ★選擇遙控
    • 打開主機體電控箱，切至遙控位置。
    • 使用遙控器:△上升 ▽下降 ■停止 ●無用。
    • △上升 ▽下降動作中要切換反向時，先按 ■停止再操作。
      ★選擇手控
    • 打開主機體電控箱，切至手動位置。
    • 主機體電控箱内開關:上、off、下，三段控制。

■充電說明:

  1. 電池:DC 12V 7hA
  2. 充電器:輸入AC 110V-240V，輸出DC 12V。
  3. 充電器插插座，紅線(＋正電)接電池正極(紅端) ，黑線(－負電)接電池負極(黑端)。
  4. 充電器顯示紅燈表示充電中，綠燈表示充電完成，充電完成請拔出插座停止充電。

■注意事項:

  1. 操作期間，勿觸碰柵欄器，避免夾傷危險
  2. 升降器遇阻礙無法正常運作時，機器會自動斷電，請先移除障礙物，在電控箱内再按下紅色按鈕使保護馬達之過載電流保護器重新通電，始可繼續操作。
  3. 遙控失效時，可切換至手控模式操作。
  4. 每組遙控與主機皆有編號，可再加購遙控器。
  5. 馬達之過載電流保護器在正常操作下，馬達保固壹年期限。
  6. 特別注意: 馬達之過載電流保護器發生斷電瀕率高時，請勿強制通電使用，請立即聯絡 本公司處理，以免馬達損壞。
